# Earicular Website

This repository contains the initial website scaffold for the BME 180C Senior Design Capstone project.

## What is included

- `index.html` — single-page website with dedicated sections for product explanation, needs statement, competitor analysis, market strategy, team info, presentation, and pitch.
- `styles.css` — responsive styling for a modern presentation layout.
- `script.js` — mobile navigation toggle and smooth scrolling for anchor links.

## How to use

1. Open `index.html` in a browser.
2. Replace placeholder team names, presentation links, and project-specific content with your final capstone materials.
3. Add your slide deck or presentation resources to the `presentation` section.

## Local editing

- Use any code editor to update the HTML, CSS, and JavaScript files.
- For local preview, open `index.html` directly or use a simple HTTP server such as Live Server in VS Code.

## Notes

- The current home page is intentionally structured around the BME 180 grading rubric.
- Customize each section to reflect your project findings, team contributions, and final pitch.
